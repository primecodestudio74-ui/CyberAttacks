## 🌌 What is HackAware?

HackAware is not just a project — it's a **cyber awareness experience** ⚡  

It allows users to explore how real-world cyber attacks work through **safe, simulated environments**, helping them understand:
- How hackers think  
- How vulnerabilities are exploited  
- How to defend against them  

---

## 🚀 Why HackAware?

💡 Because awareness is the first layer of security  
💡 Because theory alone isn’t enough  
💡 Because practice builds confidence  

HackAware transforms **learning into experience**.
